Project Information

This is a small HTML/JavaScript project utilizing multiple webpages. This purpose of this project is to design one's own custom CSS and implement 3 JavaScript enabled pages that primarily implement Objects.

This project contains the following pages:
  • Create a Payee which prompts the user for hours worked, a pay rate, and then instantiates an object with a gross pay calculation. It then prints out the object's data and the function calculation.
  • Create a Stock which instantiates 2 Stock Objects and implements functions such as getStock and getPercentChange. It then displays the output.
  • Create a Cup of Coffee which employs the use of an empty constructor and setter functions. It then prints out the contents of the object using a for loop.
